Link in body

This is a test based on this article.

In order to test this, use network throttling (slow network), to see the differences.

Explanation

Instead of placing all the styles inside the head tag, I'm splitting them in different files, by component, and placing the link tag before the html of each component.

This way, instead of waiting for every style to download, whenever a component style downloads, it renders right away.
